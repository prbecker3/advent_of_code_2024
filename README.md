## Folder Structure

This repository uses a Cargo workspace to organize multiple projects, each dedicated to a specific day of Advent of Code. The workspace structure allows for efficient building and testing of individual solutions:

```bash
advent_of_code_2024/
├── Cargo.toml
├── Cargo.lock
├── crates/
│   ├── day01/
│   │   ├── Cargo.toml
│   │   └── src/main.rs
│   ├── day02/
│   │   ├── Cargo.toml
│   │   └── src/main.rs
│   ...
│   └── day24/
│       ├── Cargo.toml
│       └── src/main.rs
```

This approach provides a clear and modular way to manage the solutions for each day.
